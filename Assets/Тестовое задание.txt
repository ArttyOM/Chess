PvP 2D Шахматы
Необходимо сделать классические двумерные шахматы без искусственного интеллекта, то есть ходить можно за обе стороны.
Необходимо использовать стандартные UI компоненты (uGUI). Должна использоваться легкая анимация, любая.
Можно использовать любую графику из интернета.
Проверяться будет стиль написания кода и общая архитектура проекта.
